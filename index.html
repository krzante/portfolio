Tarheta

Tarheta is a powerful web application designed for both students and teachers to make learning more engaging and efficient. With Tarheta, users can create flashcards for quizzes, exams, and reviewers, allowing them to review and reinforce key concepts in an interactive and enjoyable way. For teachers, Tarheta also provides the option to create classes, enabling them to organize simultaneous tests and quiz-taking for their students.

Axie Calculator App

A simple python tkinter application with axie stats API integration. Axie Calculator is an app to aid with tracking the enemy's energy. It includes features such as tracker for ranking, smooth-love-potion and winrate, persistent window and adjustable window opacity.

Into the Battlefield

Into the Battlefield is a 2D local area network multiplayer game compatible with Windows 10 laptops and Android 8.0 to 13 mobile devices. It is a thrilling game that challenges players to lead their pieces into battle and emerge victorious. The game is set in a war-torn landscape, with players taking on the role of commanders responsible for commanding their armies to victory. In this game, strategy is key, and players must make strategic decisions to outmaneuver their opponents and secure victory.
